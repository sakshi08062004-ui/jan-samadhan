# Jan Samadhan

Jan Samadhan is a fullstack web application for complaint management with admin and customer panels.

## Features
- Customer: Register, login, create complaints, view profile, feedback.
- Admin: Dashboard, view complaints, departments, analytics, feedback.

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript

## Quick Start
1. Install Python 3.8+.
2. Run backend: `python backend/fullstack_app.py`
3. Open `frontend/customer/home.html` in browser (or serve via http-server).
4. Demo data: `python backend/setup_demo_data.py`

## Start Scripts
- `START_SERVER.bat` or `start_backend.py`

Live Demo: Check LIVE_DEMO.bat
